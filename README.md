# Fight Money Dashboard

## 制作概要

Fight Moneyは、親子でお手伝いとおこづかいを管理するためのWebアプリです。

今回のページでは、JavaScriptライブラリのChart.jsを使用し、
お手伝い回数をグラフで分かりやすく表示しました。

## 制作した目的

子どもの頑張りを数字やグラフで見えるようにすることで、
お手伝いを楽しく続けられる仕組みを作ることを目的としています。

## 使用技術

- HTML
- CSS
- JavaScript
- Vite
- Chart.js

## 主な機能

- 今月のおこづかい金額の表示
- お手伝い回数の表示
- Chart.jsを使ったグラフ表示
- スマートフォンに対応した画面デザイン

## 使用したライブラリ

Chart.js

Chart.jsを使用し、曜日ごとのお手伝い回数を棒グラフで表示しています。
